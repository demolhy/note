## 深拷贝(深入版)
多层对象需要递归解决
````
function clone(target) {
  if(typeof target === 'object'){
    let cloneTar = Array.isArray(target) ? [] : {} // 对象包含数组
    for(const key in target) {
      cloneTar[key] = clone(target[key])
    }
    return cloneTar
  } else {
    return target
  }
}
````
